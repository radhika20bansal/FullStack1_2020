Welcome to FullStack Course

About course

FullStack course consist of two major aspects:
1.Web Designing which is known as Front-End Development
2.Web Developing which is known as Back-End Development
In this semester, we will study front-end development which is typically consist of Hyper Text Markup Language(HTML), Cascading Style Sheets(CSS), Javascript, jQuery, Bootstrap. It encompasses everything from creating a simple HTML text to creating complex, responsive websites designed to be accessed by different web browsers, devices and sceen-sizes.

List Of Practicals

1. Create a webpage including Html basic structure using Online Sandbox Environment and Html Editor Practical 1
2. Live your first webpage on Git-Hub Practical 2
3. Create a webpage showing a basic html structure using character entities and span tag Practical 3
4. Create a webpage having list of all practicals with their individual practical page links Practical 4
